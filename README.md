# 🧭 Navvi Explorer

O **Navvi Explorer** é uma aplicação móvel moderna desenvolvida em Flutter para entusiastas de viagens. A aplicação permite descobrir hospedagens, atividades e pontos turísticos em tempo real, consumindo dados oficiais da API Open Data Hub.



## ✨ Funcionalidades

- **Autenticação Segura:** Tela de login personalizada com validação de credenciais.
- **Carrossel Dinâmico:** Destaques automáticos na Home que mudam a cada 3 segundos.
- **Integração com API:** Consumo de dados reais da Open Data Hub (Südtirol/Bolzano).
- **Categorização:** Filtros para Hospedagens, Atividades, Cultura e Pontos de Interesse.
- **Design Premium:** Interface baseada em degradês roxos, sombras suaves e tipografia elegante.
- **Responsividade:** Layout adaptável para diferentes tamanhos de ecrã.

## 🛠️ Tecnologias Utilizadas

- [Flutter](https://flutter.dev/) - Framework de UI.
- [Dart](https://dart.dev/) - Linguagem de programação.
- [Open Data Hub API](https://opendatahub.com/) - Fonte de dados de turismo.
- [Timer & PageController](https://api.flutter.dev/flutter/widgets/PageController-class.html) - Para animações automáticas.
